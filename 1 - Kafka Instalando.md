# Instalação do KAFKA

## Prerequisitos
* Ter o Java instalado
    `$ java -version`

## Instalação
* Download do binario :: https://kafka.apache.org/downloads 

* Descompactar o arquivo
    * tar zxf <caminho-do-arquivo>

* Start Zoo

* Start Kafka
`bin/kafka-server-start.sh config/server.properties`
